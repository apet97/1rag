# CAKE.com Bundle Subscription

CAKE.com Bundle is a comprehensive package that offers access to Enterprise subscription plans for all three essential productivity solutions: **[Clockify](https://clockify.me/)**, **[Pumble](https://pumble.com/)**, and **[Plaky](https://plaky.com/)**. With the CAKE.com Bundle, you can improve your workflow and  integrate time tracking, team messaging, and project management into one cohesive suite. 

CAKE.com Bundle subscription plan includes all the Enterprise features from all the CAKE.com apps. Check out [this article](https://cake.com/help/billing/cake-com-subscription-plans-and-features/) to learn more about the features included in the Bundle plan.

## Purchase CAKE.com Bundle Plan

CAKE.com Bundle subscription can be purchased from each individual app, by following the steps for subscription purchase and choosing CAKE.com Bundle plan. Check out help articles to find out how to purchase bundle plan from each app:

* [Clockify](https://clockify.me/help/administration/subscribe-upgrade)
* [Pumble](https://pumble.com/help/workspace-administration/pricing/upgrade-a-free-workspace-to-a-paid-plan/)
* [Plaky](https://plaky.com/help/administration/billing/upgrade-a-free-workspace-to-a-paid-plan/)

When you buy one **Bundle seat**, you can invite user to use all three apps. Check out [this article](https://cake.com/help/billing/bundle-seats/) for more information about bundle seats.
